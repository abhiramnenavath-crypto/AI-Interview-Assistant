# AI-Interview-Assistant
AI-powered interview assistant built with Python that asks questions, analyzes responses, and provides feedback using voice interaction and basic NLP techniques.

##  Overview
This project is an AI-based interview assistant that:
- Asks interview questions
- Analyzes answers
- Gives feedback on confidence
- Uses voice interaction

## Technologies Used
- Python
- OpenCV
- FER (Emotion Detection)
- SpeechRecognition
- pyttsx3

## Features
- Voice-based questions
- Answer evaluation
- Confidence scoring
- Emotion detection (upcoming)

## Current Version
This version of the AI Interview Assistant includes:

- Voice output for asking interview questions  
- Answer analysis based on response length  
- Feedback system to evaluate answers  
- Confidence scoring mechanism
